# Lab 3 - Matrix and Vector Operations

## Project Description
This project is a console application designed to perform matrix and vector addition operations using Object-Oriented Programming (OOP) principles. The application allows users to input matrices and vectors, ensures the dimensions are compatible, and then performs the necessary addition.

## Features
- **Matrix Addition**: Adds matrices of the same dimensions.
- **Vector Addition**: Adds vectors of the same size.
- **Dimension Validation**: Validates that the dimensions of the matrices and vectors are compatible before performing addition.

## How to Run
1. Download the repository.
2. Open the project in **Visual Studio** or any compatible C# IDE.
3. Build and run the application.
4. Follow the on-screen instructions to input matrices and vectors.

## Example Usage
- Input two matrices or vectors.
- The application will display the result of the addition.
